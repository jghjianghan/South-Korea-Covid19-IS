# South-Korea-Covid19-IS
A web-based information system for Covid-19 cases in South Korea. This project is made as an assignment for Project Management Course in Parahyangan Catholic University.

# Built with
- HTML
- CSS (bootstrap)
- Javascript
- PHP
- Apache
- MariaDB
